### Projects

**[cylinder-booking-system](https://github.com/RemyAthisayaa17/cylinder-booking-system)**
Full-stack LPG cylinder booking and delivery management system. Covers the complete order lifecycle — UPI + cash payments, OTP-verified delivery, partner auto-assignment, refund processing, and invoice generation across Customer, Delivery Partner, and Admin roles.
`TypeScript` · `Node.js` · `React` · `PostgreSQL` · `Prisma`

NLP-based system that matches PDF resumes to job descriptions. Produces an alignment score, derived skill list, competency breakdown, and strategic recommendations using semantic matching and Gemini API.
`Python` · `Flask` · `React` · `pdfplumber` · `Gemini API`
